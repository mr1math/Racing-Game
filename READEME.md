# لعبة السيارات Racing Game

لعبة سيارات ثلاثية الأبعاد مبنية باستخدام Three.js وJavaScript.

## المميزات
- سباقات سيارات متعددة
- مسارات متنوعة
- نظام فيزياء واقعي
- واجهة مستخدم تفاعلية

## التقنيات المستخدمة
- Three.js للرسوميات
- Cannon.js للفيزياء
- Howler.js للأصوات
- Webpack للتجميع

## التنصيب
```bash
git clone https://github.com/username/racing-game.git
cd racing-game
npm install
npm start
